# Amazon Sales Performance & Operations Dashboard
Successfully completed a complex Amazon Sales Analysis project involving 100k+ rows of data, delivering insights on Revenue Trends, Customer Demographics, and Order Fulfilment using Power BI and DAX
## Project Overview
This Power BI dashboard provides an end-to-end analysis of Amazon sales data, focusing on revenue trends, product performance, and logistics operations. It is designed to help e-commerce managers make data-driven decisions regarding inventory and regional marketing.
## Key Business Insights
- **Revenue Trends:** Analyzed sales fluctuations over time to identify peak performance periods.
- **Product Performance:** Categorized sales by product type (e.g., Kurta, Set, Western Wear) to find the most profitable segments.
- **Size Analysis:** Visualized customer preferences for different sizes (S, M, L, XL, etc.) to optimize stock levels.
- **Operational Health:** Monitored the ratio of Shipped vs. Cancelled orders to improve delivery success rates.
- **Geographical Distribution:** Used map visuals to pinpoint top-performing states and cities across India.
- ## Technical Skills Demonstrated
- **Data Cleaning:** Transformed raw Amazon CSV data using **Power Query**.
- **Advanced DAX:** Created measures for **Average Order Value (AOV)**, **Total Revenue**, and **Success Rate %**.
- **Data Visualization:** Built an interactive UI using Map visuals, Donut charts, and Area charts.
- **Interactive Features:** Implemented Slicers for Date, Category, and B2B segments for a dynamic user experience.
